# Johar Clash — Lahore Gaming Tournament (landing page)

Static single-page site for a LAN gaming event in Johar Town, Lahore.
FIFA · Tekken 5 · Valorant · Call of Duty · Mortal Kombat · PUBG Mobile — PKR 25,000 prize pool.

- `index.html` — the whole site (self-contained; inline CSS/JS).
- `assets/` — hero banner, venue and game-mode art (AI-generated).
- `downloads/` — venue image, event poster, rulebook (drop real venue photos here too).
- Registration form posts to the ecommerce Apps Script API (order + telemetry).
